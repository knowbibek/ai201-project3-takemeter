# Labeling cheat-sheet (label any comment in ~3 seconds)

Run each comment through these THREE questions in order. Stop at the first answer that fits.

```
1. Does it make a CLAIM/JUDGMENT about the game, a player, or a team?
      NO  -> reaction          (it's just a feeling, joke, or vibe)
      YES -> go to 2

2. Does it EXPLAIN the claim with specific evidence or a mechanism
   ("X because Y", a stat used to reason, a tactical cause)?
      YES -> analysis
      NO  -> go to 3

3. It's a confident opinion with no real reasoning.
                                -> hot_take
```

## 2-second tells

| If you see…                                              | Label      |
|----------------------------------------------------------|------------|
| emojis, ALL CAPS, "scenes", "I'm crying", a pure joke    | `reaction` |
| "X is overrated/finished/the best", "we'll never win"    | `hot_take` |
| "because", "drops between", "the reason is", real cause  | `analysis` |
| a single stat thrown at a verdict, no mechanism          | `hot_take` |
| a stat that's actually part of an argument               | `analysis` |

## Resolved borderline cases (your reference for the CHECK rows)

1. *"Mbappé is washed — 1 goal in his last 4 knockout games."*
   One cherry-picked stat backing a verdict, no mechanism → **`hot_take`**.

2. *"People blaming the keeper are wrong — he's screened until the ball is past him; the CB
   steps out and opens the lane."*
   Explains a mechanism with specific evidence → **`analysis`**.

3. *"This is the worst defending I've EVER seen, drop them all 🤬"*
   Survives as a standalone opinion without the emoji/caps → **`hot_take`** (not reaction).

4. *"WHAT A GOAL I'M SHAKING"* → pure feeling → **`reaction`**.

5. *"England look nervous, they always do in knockouts."*
   A claim, but no specific evidence/mechanism (vague pattern assertion) → **`hot_take`**.

6. *"They're sitting too deep — the midfield line is 30 yards behind the strikers, so every
   clearance comes straight back."* → mechanism + specifics → **`analysis`**.

> Rule of thumb for the hard `analysis` vs `hot_take` boundary: if deleting the opinion still
> leaves something that *teaches you why*, it's `analysis`. If it just leaves a verdict (or a
> lone stat), it's `hot_take`.
